# VerificationOption

Determines the verification option to use for the account registration (Enum: SMS, EMAIL, CALL, INVOICE).

## Example Usage

```csharp
using Shippo.Models.Components;

var value = VerificationOption.Sms;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `Sms`     | SMS       |
| `Email`   | EMAIL     |
| `Call`    | CALL      |
| `Invoice` | INVOICE   |
